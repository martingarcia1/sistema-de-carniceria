El presente documento describe de manera clara y accesible las funcionalidades que tendrá el sistema de gestión diseñado especialmente para su carnicería. El objetivo es que usted, como cliente, pueda revisar, confirmar y aprobar el alcance del trabajo antes de que comience el desarrollo. Por favor, lea con detenimiento cada sección. Si algo no refleja exactamente lo que usted necesita, este es el momento ideal para conversarlo y ajustarlo.  1. Ingreso de Mercadería Este módulo permite registrar todo lo que entra al negocio: cortes, piezas enteras, insumos para elaboración, etc. ¿Qué podrá hacer? • Registrar productos con nombre, cantidad en kilogramos y precio de compra por kg. • Ingresar piezas enteras (por ejemplo: media res de $1.068.000 / 80 kg) y el sistema calculará automáticamente el costo por kilogramo ($13.350/kg). • Distinguir entre materia prima (carne, grasa, condimentos) y productos elaborados. • Consultar el historial de ingresos por fecha.  2. Ventas y Egresos Cada vez que se realice una venta, el sistema la registrará y descontará automáticamente el stock correspondiente. ¿Qué podrá hacer? • Registrar ventas indicando producto, cantidad en kg y precio de venta. • El stock se actualizará automáticamente tras cada venta. • Consultar el historial de ventas filtrado por fecha o producto. • Ver el total vendido por día, semana o mes.  3. Elaboración de Productos Este es uno de los módulos más importantes: permite registrar cómo se fabrican los productos elaborados a partir de materia prima, y actualiza el stock de manera inteligente. Productos contemplados:

Métricas y Ganancia El panel de métricas le dará una visión clara del rendimiento de su negocio en tiempo real. Información disponible: • Ganancia por producto: precio de venta menos precio de costo. Ejemplo: chorizo comprado a $7.500/kg, vendido a $10.000/kg → ganancia $2.500/kg. • Porcentaje de rendimiento por corte (ejemplo: carne 22%, rebo 40%). • Stock actual en kg por cada producto. • Resumen de ingresos, egresos y ganancia neta por período.  ¿Qué NO incluye este sistema (en esta versión)? Para mantener el proyecto claro y manejable en esta primera etapa, las siguientes funcionalidades quedan fuera del alcance inicial: • Caja / facturación electrónica (AFIP). • Integración con impresoras fiscales o de tickets. • Gestión de clientes o cuentas corrientes. • Aplicación para celular (puede incorporarse en una segunda etapa). Estas funcionalidades pueden incorporarse en versiones futuras si así se acuerda.  Conformidad del Cliente Si está de acuerdo con el alcance descripto en este documento, por favor firme a continuación. Esto nos permitirá dar inicio al desarrollo del sistema.
